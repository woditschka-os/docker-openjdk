# OpenJDK Docker Image

- OpenJDK 8
- OpenJDK 11

Hosted on [quay.io](https://quay.io/repository/woditschka/openjdk?tag=latest&tab=tags) 
```
docker run -it quay.io/woditschka/openjdk:8
docker run -it quay.io/woditschka/openjdk:10
```
